# **1. Какую задачу решает продукт?**

Продукт решает информационную и экономическую проблемы, а именно ускоряет создание бренд-бук ориентированных изображений. Улучшает текстовое описание пользователя, генерирует изображения в нужном стиле, оценивает их качество и выбирает лучший вариант. Следовательно, снижает трудозатраты и делает процесс создания визуалов быстрее.

---

# **2. Целевой пользователь**

## **Сегмент 1: Дизайнеры**
Данный продукт позволяет ускорить работу, быстро получать прототипы соблюдая фирменный стиль.

## **Сегмент 2: Малый бизнес/SMM специалисты**
Данный продукт позволит предпринимателям получить визуал без найма дизайнера на полную ставку.

# **3. Ценность продукта**

## **Что получаеют дизайнеры (ЦА1)**
- Ускорение рутинных этапов работы (поиск стиля, подготовка промтов)
- Набор визуалов, из которых они сами выбирают лучший
- Возможность использовать сервис для получения концепт изображений

### **Как измерить ценность**
- Снижение трудозатрат (цель - 30%)
- Удовлетворённость пользователей - NPS ≥ 7

## **Что получает пользователь ЦА2**
- Быстрое создание изображений (минуты вместо часов)
- Соответствие фирменному стилю
- Снижение ручной работы (автоматический выбор лучшего изображения)

### **Как измерить ценность**
- экономия бюджета (цель - 70-80%)
- Удовлетворённость пользователей - NPS ≥ 7

### Подписка (Subscription)

- Безлимитная генерация

- Ручной выбор изображений

- Предыдущий уровень подписки

#### Pro - 3990 ₽/мес
